About & Mentorship

Front-End Developer — Turning Ideas into Real Interfaces

As a passionate front-end developer with a strong foundation in HTML, CSS, and JavaScript, I craft clean, accessible, and responsive web interfaces. My background in full stack development allows me to collaborate effectively across teams, but my true focus lies in creating intuitive user experiences and visually polished front-ends.

Through academic and personal projects, I've built everything from single-page applications with React and GraphQL to MVC-style websites with secure image uploads and REST APIs. Whether it's implementing XSS protection, structuring semantic HTML for SEO, or connecting to a real GraphQL backend, I strive to write maintainable code with users in mind.

Curious by nature and self-driven, I thrive in collaborative environments where quality, autonomy, and creativity matter. For me, being a developer means more than writing code — it's about building tools that solve real problems and delight real people.

Technologies I love working with

HTML5, CSS3, JavaScript, React, GraphQL, Git, and a growing appreciation for UI/UX design.
